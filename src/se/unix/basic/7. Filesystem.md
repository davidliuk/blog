### ACL

可以给每一个用户一个独一无二的权限（也可以给一个目录）



挂载的时候，打开ACL选项（好用，但是开销比较大）



```shell
getfacl <file name>
 -m u::# 修改权限
 -x # 删掉一个权限
setfacl 
```



ACL mask给一个最大的权限，所有人的权限不超过这个



copy ACL

给一个目录设置一个ACL权限，然后下面所有的文件都有权限

```shell
setfacl -dm u: # 多了一个d，可以让所有人都可以
```
