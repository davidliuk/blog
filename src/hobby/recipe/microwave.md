# Microwave Oven



不能进的

- 金属
- 葡萄和类似大小的东西
- 鸡蛋



- 薯片
- 葱姜蒜泥
- 焦糖



## 蒸蔬菜

| **品类**   | **瓦数** | **时间 (分钟)** | **处理方式**                   |
| ---------- | -------- | --------------- | ------------------------------ |
| **空心菜** | 800W     | 2-3             | 洗净后切段，不需要加水         |
| **芥兰**   | 800W     | 3-4             | 洗净后切段，加少量水 (约2汤匙) |
| **小白菜** | 800W     | 2-3             | 洗净后切段，不需要加水         |
| **菠菜**   | 800W     | 2-3             | 洗净，不需要加水               |
| **油菜**   | 800W     | 2-3             | 洗净后切段，加少量水 (约2汤匙) |
| **茼蒿**   | 800W     | 2-3             | 洗净后切段，不需要加水         |
| **西兰花** | 800W     | 4-5             | 切成小块，加少量水 (约2-3汤匙) |
| **胡萝卜** | 800W     | 5-7             | 切成片或小块，加少量水         |
| **土豆**   | 800W     | 7-10            | 切成小块，加少量水，或整颗     |
| **绿豆**   | 800W     | 3-4             | 加少量水                       |
| **青豆**   | 800W     | 3-5             | 加少量水                       |
| **西葫芦** | 800W     | 3-4             | 切成片或小块，加少量水         |
| **西红柿** | 800W     | 2-3             | 切成小块或片，不需要加水       |
| **茄子**   | 800W     | 5-6             | 切成片或小块，加少量水         |
| **玉米**   | 800W     | 6-8             | 去皮后整根放入，不需要加水     |
