### 项目

#### 雁过留痕

雁过留痕 Trace Note 是一款基于地点的社交 app。用户可以在一个地点开启一个 Trace AR 场景，并在场景中放置 Note 虚拟留言板，不同用户见可以在同一个 Trace 中相互交互评论。Note 留言板提供字体、材质等个性化装饰选项。用户可以通过连续签到打卡获取代币，用来抢购个性化装扮，以提高个性化程度，并作为激励机制提高用户每日的活跃度。

#### 仿掘金官网

一款类掘金的知识分享平台。我们利用 TrustSVD 社交推荐算法，实现了基于用户个人偏好及社交关系的个性化推荐系统。用户可以发布文章、收藏文章、评论互动（隐式）、关注作者，以及获得个性化推荐结果。

### 过往遇到的比较有挫折感的事儿

在我去年暑假期间，我参加了移动应用创新赛，我们的团队致力于开发一个综合型的AR社交APP。我们设定了高目标，希望能够在规定的时间内完成一个功能丰富且稳定的应用。

然而，我们在开发过程中遇到了一系列的挑战。首先，我们的团队在初期对项目的范围和技术要求有些过于乐观，导致我们在开发过程中遇到了时间紧迫和资源不足的困境。

其次，我们遇到了一些技术难题。由于项目要求较高，我们需要掌握一些新的技术和框架。然而，我们在学习和应用这些新技术时遇到了困难，并且在某些功能的实现上遇到了一些 bug 和兼容性的问题。

在比赛的中期阶段，我们感到非常沮丧和挫折。项目进展不如预期，而且时间越来越紧迫，难以完全实现项目立项时设立的愿景。

然而，我们没有放弃，而是通过团队会议和讨论来共同寻找解决方案。

我们决定重新评估项目的优先级和功能范围，将实现的重点放在核心功能的实现上，并推迟了一些次要功能。我们与小组的指导导师和其他同学进行交流，并得到了一些宝贵的建议和指导。我们也积极调研市面上成熟的解决方案，寻求其他开发者的帮助和经验分享。

最终，在比赛截止日期前的最后几天，我们成功地完成了核心功能并解决了大部分 bug。虽然我们没有完全实现最初设想的所有功能、也并没有斩获一等奖的成绩，但我们应用的核心功能在比赛中获得了评委的好评，并取得了不错的成绩。

通过这个挫折的经历，我学到了很多关于团队合作、项目管理和技术应用的重要教训。我学会了灵活调整项目计划和目标，同时善于寻求他人的帮助和指导。这个经历使我更加注重解决问题的能力和适应能力，同时也加深了我对软件开发过程中团队协作的理解。

面对这个问题，你可以采取以下步骤：

1. 诚实回答：你可以谈论一个真实的、有挑战的情境，如何应对它以及从中学到的教训。HR 面试官通常会更欣赏你的诚实，而不是一个完美无瑕的表现。
2. 关注解决问题的方法：不要过多强调失败本身，而是强调你如何应对和解决挑战。描述你在这个情境中采取了什么措施，如何克服困难，并讲述你的策略是如何帮助你克服障碍的。
3. 强调学习经验：不要让负面情境令你感到沮丧，而是着重于你从中学到的经验和教训。谈论你从这个经历中学到了什么，以及你如何在未来应对类似情境。

举例来说，你可以这样回答：“在我的上一份工作中，我遇到了一个项目延期的情况。我首先尝试了多种不同的解决方案来加速项目进程，但都没有成功。然后，我意识到我们需要重新评估时间表，并与客户进行沟通以重新协商时间表。我认为，这个经历教会了我需要更加灵活地应对挑战，并与团队和客户进行沟通和协商。”

总之，对于这个问题，诚实、关注解决问题的方法和强调学习经验都是很好的回答策略。

2. 音乐和艺术：讲述你如何通过音乐和艺术来放松。你可以提到欣赏音乐、学习乐器、绘画、摄影、看电影或者参观艺术展览等活动，这些可以帮助你沉浸于创造性的世界中，放松身心并激发灵感。
3. 阅读和学习：谈论你如何通过阅读和学习来放松并扩展自己的知识。你可以提到喜欢阅读书籍、报纸、杂志或者学术文章，以及参加在线课程或者学习新的技能，这些活动可以帮助你放松同时保持个人成长。
4. 旅行和探索：强调你如何通过旅行和探索来放松和开阔眼界。提到你喜欢探索新地方、体验不同文化、品尝美食、与朋友共度时光或者远离城市喧嚣的自然之旅，这些活动可以帮助你放松并获得新鲜感。

无论你选择哪种方式放松，重要的是展示你有意识地寻找平衡，关注自身的心理健康和个人成长。强调你的解压方式是积极、健康和有益的，并与面试官分享你通过这些方式如何充实自己并保持积极心态。

### 平衡学业和竞赛/科研

平衡校园活动、科研和竞赛需要有效的时间管理和组织能力。以下是一些建议来平衡这些方面：

1. 制定优先级：确定我对校园活动、科研和竞赛的优先级顺序。我会根据任务的紧急程度和重要程度，划分为重要紧急、重要不紧急、不重要紧急和不重要不紧急四类。这样可以帮助我在有限的时间内更好地分配资源。

   > 根据你的个人目标和兴趣，确定你最重视的活动，并将其放在首位。

2. 利用进度管理工具：我有使用Trello来规划、标记和跟踪任务。确保合理安排每天的时间。我采用任务池法来管理日常任务，包括计划、就绪、执行、验收、完成、阻塞共六个池，以每周为单元进行任务的复盘验收，确保每项重要任务都按时保质保量的完成，包括专注于科研工作、参加校园活动以及备战竞赛。

3. 设定目标和计划：为每个方面设定明确的目标，并制定相应的计划。在每学期或每个阶段初期，制定一个时间表或日程安排，确保给予每个方面足够的时间。同时，将科研和竞赛项目与校园活动相结合，以最大程度地充实我的时间。

4. 优化学习和工作环境：创建一个有利于学习和工作的环境，减少干扰因素。寻找一个安静的地方专注于科研工作，并为参与校园活动和竞赛提前规划和准备。

5. 寻求支持和合作：寻求他人的支持和合作是平衡的关键。与导师、同学或团队成员交流，并寻求他们的建议和帮助。共享任务和分担工作负担，这样可以减轻个人压力并提高效率。

6. 保持身心健康：不要忽视身心健康的重要性。合理安排休息时间，进行适当的锻炼、睡眠和放松活动。保持身心健康可以增强你的学习和工作效率，使你能够更好地平衡各个方面。

### 比赛困难解决


在之前的比赛中，我们团队面临了几个主要的困难，但我们通过团队合作和创造性的解决方法成功地克服了它们。

首先，我们在比赛开始时遇到了一个困难，即在限定的时间内完成复杂的开发任务。我们的项目涉及多个模块和功能，而时间非常有限。为了解决这个问题，我们紧密合作，制定了详细的项目计划，并根据每个阶段的重要性和紧急性来分配任务和优先级。我们还通过定期的团队会议和进度跟踪，确保每个人都明确任务目标，并及时协调解决问题。

其次，我们在开发过程中遇到了技术挑战。由于比赛要求使用新的技术和框架，我们团队中的一些成员对其不太熟悉。为了解决这个问题，我们积极学习和掌握新技术，参考官方文档和教程，以及与其他团队成员进行经验分享。我们还与导师和其他专业人士进行沟通，并在需要时寻求他们的指导和建议。通过共同努力和学习，我们逐渐解决了技术挑战，并成功地应用到我们的项目中。

另外，我们在比赛中也遇到了一些沟通和协调方面的困难。由于团队成员之间的日程安排和个人差异，有时很难找到统一的时间进行讨论和决策。为了解决这个问题，我们采用了灵活的沟通方式，如在线聊天和远程会议，以方便团队成员之间的交流和讨论。我们也建立了有效的项目管理工具和协作平台，以便团队成员能够共享进展和资源，并及时解决沟通障碍。

总体而言，我们团队通过密切的合作、持续的学习和灵活的解决方法克服了在比赛中遇到的困难。这些挑战不仅提醒我们团队的重要性，也加深了我们对协作、沟通和技术应用的理解和能力。通过共同努力和不断学习，我们成功地完成了项目，并取得了良好的成绩。

### 将来就业会考虑什么类型公司

1. 文化和价值观相符的公司：我会优先考虑与个人价值观和兴趣相契合的公司。公司重视社会责任和科技向善，并且注重员工发展、工作生活平衡、多元化和包容性。
2. 行业领导者：我会倾向于加入在所选择的行业中具有领导地位和市场影响力的公司。这些公司通常拥有先进的技术和创新能力，并能够为我提供更多的职业发展机会和挑战，并且可以让我参与到对社会有益的产品的研发和设计。
3. （创业公司：如果你对创新和创业有浓厚的兴趣，你可能会考虑加入初创公司。这些公司通常具有灵活的工作环境和较小的团队规模，提供更多的机会去发挥创意和影响力。你也可以在早期参与到公司的成长和成功中。
4. 大型跨国公司：加入大型跨国公司可能意味着更广阔的发展机会和全球资源的利用。这些公司通常具有完善的培训计划和晋升机制，能够提供稳定的职业发展路径和较高的福利待遇。）

在回答这个问题时，可以结合你的兴趣、价值观和职业目标来说明你对某种类型的公司的倾向性，并解释为什么你认为这种类型的公司更适合你的职业发展。重要的是要展现出你对公司的研究和了解，并能够将自己的目标与公司的特点和文化相匹配。
