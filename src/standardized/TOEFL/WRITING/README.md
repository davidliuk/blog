### 独立写作得分点

- 思路，切题

- 结构，固定

- **展开**，快速提分点
- 逻辑，连贯与衔接（重点），一定要每句话都写清楚，语料中强调

上面四个部分都做到了，就可以 25+，影响大的就是展开思路（这个需要背语料）
