# 文章结构

### 文章结构

看到问题想解决方案之后有负评
现象出现找因果多重并列稳拿分
观点对比找焦点陈日观点未必错
众所周知有转折全盘否定或补充
科技文章顺因果实验文章找结论

### 做题速度

稿纸拿下立刻标 5 分 3 题不慌乱
每段结束写中心 题目不要超 3 分
走投无路深呼吸 中心态度和尾句

### 复习方法

1. 8 篇文章二次精读
   1. 逐句翻译
   2. 主千
   3. 逻辑和逻辑要素
   4. 句子之间的关系
   5. 和 TS 中心的关系
   6. 选项错误分析
2. 必考词 24Iist+同义替换
3. 句子-拆句-助教改 20 个句子
4. 单项课
5. 做题-精读
