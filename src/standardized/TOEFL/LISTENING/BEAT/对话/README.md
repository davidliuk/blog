# 对话

考点：对比、因果、问题解决方案、==结尾15s==

对话有4-6个问题诊断（Layer），每个诊断关注4点：

1. 问题是什么

2. 解决方法是什么

3. 不能解决的原因是什么

4. 有没有附加的变通解决方案

★ 重点关注出题逻辑信号：对比、因果 



来意

1. 开头背景铺垫后
2. 结尾
3. 自己总结



1. 对比
   - 比较级
   - 转折，A ..., but **B ...**
2. 因果关系
