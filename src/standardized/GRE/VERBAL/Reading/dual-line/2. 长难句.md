常用于：阅读第一句话、题目题干

### 状语

#### 顺：

①前因 :

原因: due to, because of, as a result of, because, since,as,for
方式: by, though.with
条件: if when, once , whenever

②后果

结果: so as to, so that, so
目的: in order to, to, in order that

③伴随
(with) doing



#### 反:

让步: despite, in spite of, although, though, even though/as/if, while
