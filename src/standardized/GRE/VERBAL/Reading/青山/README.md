## 双线阅读法

1. 功能线：兜底，决定分数下限

   - 单句（视角）
- 句间（句间关系）
   - 全文（全文结构）
   

利用**视角、逻辑词**，读出

1. 上下文具体作用（具体功能）
   2. 是否影响文章主旨（宏观功能）

2. 信息线

   灵活

   带着预测，有意识的寻找，修正、补充、反推信息，最终验证已经被给出的逻辑关系。
