# Seq2seq

Input a sequence, output a sequence. The output length is determined by model.

- Speech Recognition 
- Machine Translation
- Speech Translation
- Text-to-Speech (TTS)
- Question Answering (QA) (ChatBot)
- Multi-label Classification
