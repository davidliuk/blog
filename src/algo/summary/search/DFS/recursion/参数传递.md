### 值传递

值传递相当于创建个新的变量，把值拷贝一份

修改新的变量原来的变量不会受到影响

---

不同语言中呈现值传递的场景

Java的基本数据类型

(byte,short,int,long,float,double,char,boolean)

C++中默认值传递

Python没有值传递

基本数据类型的值传递

---

Python 没有值传递

但是 Python 的不可变类型可以认为是值传递的

### 引用传递

可以理解为别名，代号

文件的快捷方式

所谓引用传递是指在调用函数时将实际参数的地址传递到函数中，那么在函数中对参数所进行的修改，将影响到实际参数。

Java 除基本数据类型以外的其他数据

C++ 中在参数列表中加地址符 & 修饰

Python 中全是引用传递

### 修改实例和修改引用

带 “.” 或者带 “[]” 修改的是实例

否则修改的是引用

包装类和不可修改类型的传参

虽然是引用传递，但是能看成是值传递
