### 动归四要素

动规的方程Function --- 递归的拆解

dp\[i][j] = dp\[i-1][j] + dp\[i][j-1]

```java
void uniquePath(int m, int n) {
    // state: dp[i][j] 代表0, 0走到i, j的方案数
    int[][] dp = new int[m][n];
    
    // init
    for (int i = 0; i < m; i++) {
        dp[i][0] = 1
    }
    for (int j = 0; j < n; j++) {
        dp[0][j] = 1
    }
    
    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
        }
    }
    
    return dp[m - 1][n - 1];
}
```
