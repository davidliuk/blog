有序数组

去重：

```java
if (j < n - 1 && nums[j] == nums[j + 1]) {
    // 跳过所有与i重的元素
    // 保留最后一个，在双指针的时候可以枚举到唯一一对相同值的数对
    continue;
}
if (i > 0 && nums[i] == nums[i - 1]) {
    // 跳过所有与i重的元素
    // 保留第一个，这种写法就是组合II那种去重思路
    continue;
}
```
