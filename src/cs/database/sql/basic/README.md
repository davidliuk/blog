# 语法

变量声明

## 空表

无合适答案（空表）的时候，返回null的方法

需要嵌套子查询

### 空表格

空表示表格没有任何输出，null表示输出为null

| 入口   | 空表格 |
| ------ | ------ |
| SELECT | null   |
| FROM   | 空     |
| LIMIT  | 空     |
| SUM    | null   |
| AVG    | null   |
| MAX    | null   |
| MIN    | null   |
| COUNT  | 0      |
| IFNULL | 空     |
| 窗口   | null   |
