# 其他引擎及选择



### MyISAM

MyISAM是MySQL早期的默认存储引擎

#### 特点

- 不支持事务，不支持外键
- 支持表锁，不支持行锁
- 访问速度快

#### 文件

- myd 存放数据

- myi 存放索引

- sdi 表结构




### Memory

Memory引擎的表数据时存储在内存中的，由于受到硬件问题、或断电问题的影响，只能将这些表作为临时表或缓存使用。

#### 特点

- 内存存放
- hash索引（默认）

#### 文件

- xxx.sdi: 存储结构信息
