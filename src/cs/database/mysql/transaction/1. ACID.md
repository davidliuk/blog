# 事务特性：ACID

## 四大特性

- 原子性（Ato'micity）

  事务是不可分割的最小操作单元，要么全部成功，要么全部失败（并回滚）。

- 一致性（Con'sistency）

  事务完成时，必须使所有的数据都保持一致状态。

- 隔离性（Isolation）

  数据库系统提供的隔离机制（隔离级别），保证事务在不受外部并发操作影响的独立环境下运行。

- 持久性（Dura'bility）

  事务一旦**提交或回滚**，他对数据库中数据的改变就是永久的。

> MyISAM 引擎不支持事务，InnoDB 引擎通过多种机制实现了 ACID
