# Hash 类型

- `hset` 设置 key 的一个或多个 field

- `hget` 获取 key 的一个 field

- `hmset` 设置 key 的一个或多个 field
  > hmset 已经废弃了，可以直接用 hset 设置多组键值对了
- `hmget` 获取 key 的一个或多个 field

- `hgetall` 获取所有键值对

- `hkeys` 获取所有键

- `hvals` 获取所有值

- `hincrby` 让 key 的一个 field 自增

- `hsetnx` 设置 key 的一个 field，如果 field 已经存在，则不设置
