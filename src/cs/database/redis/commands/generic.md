# 通用命令

## 键操作

- `KEYS` 查询符合模板的 key，不建议在生成环境设备上使用
- `DEL` 删除 key
- `UNLINK` 删除 key，不会阻塞客户端，Redis 4.0 之后才有的命令
- `EXISTS` 判断 key 是否存在
- `EXPIRE` 给 key 设置有效期，有效期到期时该 key 会被自动删除
- `TTL` ：查看一个 KEY 的剩余有效期
  - -1：永久有效
  - -2：过期了

## 设置操作

- `SET` 设置 key 的值

## key 的层级格式

用":"来分隔，形成层级

一般格式：

- \[项目名\]:\[业务名]:\[类型]:\[id\]
