# DFS



DFS相对于BFS的优势：

- 后序传值（DFS没有后序）
- 先序回溯（BFS不能回溯）



栈搜索

运行时栈

入栈：函数开始运行

出栈：函数运行结束



顺序：

1. 出栈（即函数被调用了）
2. 捕捉：全局变量捕捉target值
3. 扩展
4. 入栈



### 先序遍历

从已知点开始出发

（写法在本质上与BFS非常相似，本质上还是拓扑排序）





### 后序遍历
