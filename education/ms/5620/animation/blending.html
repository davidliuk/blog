Problem ‚Äì After playing an animation with root translation, the root joint will snap back to the initial position and orientation after playing the animation

Solution ‚Äì Add a virtual joint called a ‚ÄúGuide‚Äù as a parent of root joint ‚Äì As a result, root position and orientation transforms are now with respect to the guide frame of reference ‚Äì At the end of each animation, update the guide joint position to the current position of the character in the world

Foot IK

Allows character to walk over uneven terrain

IK Targets ‚Äì Update left and right foot target positions

  • Lfoot_target = Lfoot + Lfoot_height
  • Rfoot_target = Rfoot + Rfoot_height

Root Position

  • Determine which foot is lower
  • root_offset = smaller of Lfoot_height and Rfoot_height
  • Update vertical position of root joint
  • y_root = y_root + root_offset

Foot Position ‚Äì Use Limb IK with Lfoot_target and Rfoot_target to position feet
